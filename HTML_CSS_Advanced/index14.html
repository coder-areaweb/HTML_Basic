Мой первый текстовый абзац со стилевым оформлением

Изменить цвет можно:

  • Текст color: red
  • Фон background-color
  • С помощью зн-й: values (RGB/HEX/HSL/RGBA/HSL/HSLA)
  • С помощью зн-й: values (HWB/LCH/LAB)
  • Color names исползуя предопределенное имя цвета color: red/lightgray

Шестнадцатеричная система счисления:

  • от 0-9 #000000 (black)
  • от A-F #FFFFFF (white)
  • # означает НЕХ значение цвета
  • Цвет текст в формате 16-и системе исчесления (HEX) color: #77cc00
  • Цвет текст в формате 16-и системе исчесления сокращенной (HEX) color: #7c0
  • Цвет текст в формате RGB (Red Green Blue)/(#RRGGBB) color: rgb(37,188,57)
  • Цвет текст в формате RGBA ed Green Blue Alpha) color: rgba(37,188,57, 0.8)
  • Web-цвета зависят от цветопередачи монитора и принимают 1 из 6 зн-й: 0(00)/51(33)/102(66)/153(99)/204(СС)/255(FF)
    Может применять 0-255 оттенков R+G+B=R(256)*G(256)*B(256)=16,777,216 комбинаций

Другое использование:

  • Наименованные цвета:используются в макетах
  • RGB and HEX:верстка
  • LAB и LCH определены в спецификации как цвета, не зависящие от устройства

RGB методы:

  • Аддитивное смешение цветов - сложение цветов непосредственно излучающих объектов (смешивание лучей света)
  • Субтрактивное смешение цветов получение цвета путем вычитания из спектрально-равномерного белого света отдельных спектральных составляющих (вычитании элементов друг из друга)

HSL расшифровка (создать свой цвет или подобрать):

  • Оттенок градус цветового круга 0-360 RED(0)/GREEN(120)/BLUE(240)
  • Насыщенность (интенсивность цвета)% зн-е: 0% оттенок серого/ 100% полный цвет
  • Яркость (кол-во света, хотите дать цвету) тоже %:ЧЕРНЫЙ(%)/НИ СВЕТЛЫЙ,НИ ТЕМНЫЙ(50%)/БЕЛЫЙ(100%)
  • HSL(HUE SATURATION LIGHTNESS) Colors Оттенок Насыщенность Яркость: hsl (0,100%,50%)
  • HSLA (HUE SATURATION LIGHTNESS ALPHA) Colors

Параметр альфа - это число от 0,0 (полностью прозрачное), до 1,0 (совсем непрозрачно)

HWB

  • LAB - это цветовое пространство, доступ к которому возможен в таких программах, как Photoshop, и рекомендуется, если вы хотите, чтобы цвет на экране выглядел так же, как, скажем, напечатанный на футболке.
						Он использует три оси: яркость, за которой следуют ось a (от зеленого к красному) и ось b (от синего к желтому).
  • LCH означает яркость, цветность и оттенок.
						Как и в случае с LAB, яркость может быть в процентах(%), превышающих 100%.
						Аналогично HSL, оттенок может быть в диапазоне от 0 до 360.
						Цветность представляет собой количество цвета, и мы можем думать о ней как о чем-то подобном насыщенности в HSL

Изменение шрифта:

  • Атрибут color: p color="цвет"
  • Тэг font с атрибутом color: font {color:"цвет"}
  • Атрибут style он же Text Color: p style="color:"цвет"
  • Атрибут style в тэге HEAD: style font {color:"цвет"} style
  • Классы CSS:
  • назначаем в ф-ле сss класс: .red-text{color:цвет"}
  • в HTML указываем class в тэге: p class="red.text"
  • JavaScrip: style.color
  • Внешнего файла CSS
