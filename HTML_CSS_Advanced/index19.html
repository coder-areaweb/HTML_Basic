Блочная модель (БМ)

БМ - это правило по которому браузер определяет размер элементов на странице, его ширину и выстоту и др-е праметры.

Параметры:

  • width and height (contribution)
  • border (граница эл-та)
  • margin внутний отступ
  • padding внутренний отступ
