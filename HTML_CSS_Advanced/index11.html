Методы определения стилей (Стилизация CSS)

  • <Связывание (Linking) или ГЛОБАЛЬНЫЙ (подключение к file style.css)

    • link rel="styledheet" type="text/css" href="name.css"

  • Встраивание(Embended)

    • style type="text/css"


    p {
    color:red
    }
    style


  • Вложение(Inline)

    • style="color:name"; background="#f0f"


    p(атрибут) style color:name (декларация)


  • Импорт(Import)

    • @import url('file name css')

  • удобно для разделения файлов и правил
  • Глобальный в элементе head

    • head

    style and /style

    /head

  • Ставиться только в описании HEAD
